#  취미한다 Hobbie-Do

### 🌱 프로젝트 목표
사용자들에게 맞춤 취미를 추천해주고, 사용자들은 자신이 설정한 지역과 추천된 취미를 바탕으로 소모임 생성 및 가입을 할 수 있으며, 해당 소모임에서 게시판, 채팅 등 커뮤니티 활동을 할 수 있다

## 🏆 주요 기능
- **소셜로그인**
- **로그인 , 회원가입 - 문자인증**
- **활동지역 등록 - 카카오 맵 API**
- **취미 설문 조사**
- **소모임 생성 및 가입**
- **게시글 작성, 수정, 삭제**
- **게시글 좋아요, 댓글**
- **소모임 신청서**
- **소모임 알림**
- **소모임 단체 채팅**
